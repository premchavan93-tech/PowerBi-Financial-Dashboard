# Power BI Financial Dashboard

This repository contains a Power BI dashboard for financial analysis, including:
- Revenue and expense trends
- Profit & loss overview
- Key financial KPIs
- Monthly and yearly comparisons

## Files & Structure

- `reports/Power BI Financial Dashboard.pbix`  
  Main Power BI report file.

- `docs/dashboard-overview.png`  
  Screenshot/preview of the report.

- `data/`  
  (Optional) Sample or anonymized data used to build the dashboard.

- `config/environment-notes.md`  
  Notes on data connections, parameters, or environment setup.

## Requirements

- [Power BI Desktop](https://powerbi.microsoft.com/desktop/)  
- Access to the data source(s) used in the report (or sample data provided in `/data`).

## How to Use

1. Clone or download this repository.
2. Open `reports/Power BI Financial Dashboard.pbix` with Power BI Desktop.
3. If needed, update data source paths:
   - Go to **Transform Data → Data Source Settings**.
   - Update file paths / server / database names as required.
4. Refresh the report.
